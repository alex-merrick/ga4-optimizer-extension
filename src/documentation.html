GA4 Optimizer Documentation

GA4 Optimizer aims to solve common challenges in the Google Analytics™ 4 interface. This guide explains how each feature can improve your daily analysis, reporting, and insight-gathering tasks.

Quick Access to Settings: Most features can be toggled on/off from the extension popup. Features with more detailed configurations have a settings button next to their toggle in the popup.

1. Click to Copy Cell

Applies to: Explorations & Standard Reports
Quickly copy values from table cells, headers, or dimension labels for easier data export to spreadsheets or presentations.

2. Sticky Report Header

Applies to: Standard Reports Only
Keeps the header row visible while scrolling, making it easier to analyze long reports without losing track of metric columns.

3. Highlight Sampling

Applies to: Explorations & Standard Reports
Instantly see when your data is sampled. This feature makes the data quality icon pulse so you're always aware of potential data inaccuracies.

4. Percentage Change Highlighter

Applies to: Standard Reports Only
Uses conditional formatting to highlight significant positive or negative trends, helping you spot important changes in your performance metrics.

5. Δ% Share Change (Reports)

Applies to: Standard Reports Only
In comparison mode, this calculates the percentage point change in a metric's share of the total, providing deeper context than the standard % change.

6. Percentage of Column Total

Applies to: Explorations Only
On hover, this instantly calculates each row's percentage contribution to the column total, ideal for quick contribution analysis.

7. Auto Detailed Results (Beta)

Applies to: Explorations Only
For better data accuracy, this feature automatically attempts to request the most granular data available for your Exploration reports, a key method to reduce data sampling.

8. Percentage of All Users

Applies to: Explorations Only
Quickly calculate and compare user conversion rates for different segments against the total user base, simplifying funnel and segment performance analysis.

Requirement: The 'All Users' segment must be included in your Exploration table setup.

9. AB Test Segment Comparison

Applies to: Explorations Only
A powerful tool for conversion rate optimization (CRO) that compares metrics between test variations, calculates statistical significance, and checks for sample ratio mismatch (SRM).

Requirement: Segments must be named with `VAR-` prefix, and the control segment must start with `VAR-Control`.

10. Collapsible Panels

Applies to: Explorations Only
Get more space for your data. This adds a toggle to collapse the "Variables" and "Tab Settings" panels for a less cluttered analysis view.

11. Sticky Conversion Calculator

Applies to: All GA4 Pages
A quick-access calculator for common marketing calculations like conversion rate or CPC, available anywhere in the GA4 interface.

12. Data Range Presets

Applies to: Standard Reports Only
Speeds up your reporting by adding one-click buttons for common date ranges and comparison periods, like "Last 7 Days" or "Month-over-Month."
