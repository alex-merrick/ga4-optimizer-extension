GTM Data Layer Trigger Finder

Compare dataLayer pushes from GTM's Preview mode to find the unique parameters needed to build the perfect Custom Event trigger.

?

How This Works

  1. Enable Preview Mode in Google Tag Manager and perform the actions you want to track.
  2. In the Preview panel's event timeline, select a Custom Event (e.g., `gtm_click`).
  3. Go to the **Data Layer** tab and copy the entire block of data.
  4. Paste the data layer objects you **WANT** to trigger on into the **"Positive Examples"** box.
  5. Paste data layer objects from similar events you want to ignore into the **"Negative Examples"** box.
  6. Click "Find Unique Parameters" to generate your GTM trigger conditions.
