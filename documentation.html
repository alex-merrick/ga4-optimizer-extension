GA4 Optimizer Documentation

GA4 Optimizer aims to solve common analysis needs when using Google Analytics™ 4 interface. This guide explains what each feature does and how to use it.

Quick Access to Settings: Most features can be toggled on/off from the extension popup. Features with more detailed configurations have a settings button next to their toggle in the popup.

1. Click to Copy Cell

Applies to: Explorations & Standard Reports
Quickly copy values from table cells, headers, or dimension labels. Detailed behaviors can be configured via the settings button in the popup.

2. Sticky Report Header

Applies to: Standard Reports Only
Keeps the header row of standard reports visible at the top of the screen as you scroll down long tables.

3. Highlight Sampling

Applies to: Explorations & Standard Reports
Makes the data quality/sampling icon more noticeable by adding a subtle pulse animation when your report data isn't 100% complete.

4. Percentage Change Highlighter

Applies to: Standard Reports Only
Visually highlights cells containing percentage changes based on user-defined ranges and colors, helping to spot significant changes quickly.

5. Δ% Share Change (Reports)

Applies to: Standard Reports Only
Calculates and displays the percentage change of the "percentage of total" values (share) directly in the "% change" row.

6. Percentage of Column Total

Applies to: Explorations Only
Calculates and displays the percentage contribution of each row's value relative to the column's total on hover.

7. Auto Detailed Results (Beta)

Applies to: Explorations Only
Automatically attempts to switch Exploration reports to use "More detailed results" for better accuracy.

8. Percentage of All Users

Applies to: Explorations Only
Provides quick conversion rate insights by comparing segments against an "All Users" segment.

Requirement: The 'All Users' segment must be included in your Exploration table setup.

9. AB Test Segment Comparison

Applies to: Explorations Only
Compares performance metrics between test variations relative to a control segment, including statistical significance and SRM checks.

Requirement: Segments must be named with `VAR-` prefix, and the control segment must start with `VAR-Control`.

10. Collapsible Panels

Applies to: Explorations Only
Adds a toggle button to collapse/expand the "Variables" and "Tab Settings" panels for more analysis space.

11. Sticky Conversion Calculator

Applies to: All GA4 Pages
Provides a quick-access conversion rate calculator widget that can be opened and dragged around the screen.

12. Data Range Presets

Applies to: Standard Reports Only
Adds quick-select buttons for common date ranges and comparisons next to the date picker.
